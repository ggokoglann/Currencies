# Para Bak
Para Bak Yaptığım kaçıncı uygulama bilmiyorum ama yayınlayacağım ikinci uygulama olacak, Bu uygulamayı yaparken amacım yeteneklerimi geliştirirken aynı zamanda portfolyoma bir app daha eklemekti.
Bu app'i geliştirmeye başlarken amacım api'lerin nasıl çalıştığını öğrenmek ve aynı zamanda MVVM mimarisini kullanarak bu konudaki anlayış seviyemi de arttırmaktı.
Para Bak tek ekrana sahip ve o tek ekranında da 6 adet para biriminin anlık türk lirası değerlerini gösteren bir uygulama.
